- `tests/core/api/test_api.py` 不再承担整套 API 行为真源，只保留少量 compat smoke；
- status/command/schedule/codec/power 行为测试主要回到各自分层 test home；
- `LiproClient` compat shell 不再通过测试分布反向放大为正式入口；
- targeted API regressions 与分层测试继续通过。

- mega-file reduced to compat smoke and seam checks
- layered API tests become the primary behavior authority
- duplicated request/payload/error assertions collapsed
