## Verification Criteria

- `_ClientTransportMixin` 已从生产代码中物理移除，仓内不再存在活跃 import / export / consumer；
- `_ClientBase` 不再承担旧 skeleton / compat shell 语义，endpoint typing 改由本地 protocol port 明确表达；
- endpoint collaborators 继续通过显式 adapter / composition 工作，而不是通过 mixin inheritance 暗示第二套 API 设计；
- focused API tests 证明 endpoint behavior 未回退，且 helper tests 不再依赖 legacy mixin hierarchy。
