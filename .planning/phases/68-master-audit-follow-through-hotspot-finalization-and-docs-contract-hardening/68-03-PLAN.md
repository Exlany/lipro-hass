# Plan 68-03: keep runtime infra and runtime access thin while localizing listener and projection density

<objective>

Purpose: 落实 `ARC-15 / HOT-24 / HOT-25`，继续守住 control-plane thin projection posture。
Output: thinner runtime listener/projection helpers、focused runtime tests、dependency guards aligned with the refined seam.
</objective>
