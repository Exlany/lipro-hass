# Phase 25.2 Research

**Status:** `research complete`
**Date:** `2026-03-17`
**Requirement:** `OBS-04`, `GOV-20`

## Executive Judgment

`25.2` 的关键不是“把一个属性名改掉”，而是把 control-plane telemetry bridge 拉回到**已经正式裁决过的 runtime/protocol truth** 上，并同步 authority / residual / derived-map 叙事。

最优拆分是 `3 plans / 3 waves`：

1. `25.2-01`：在 `runtime_types.py` 与 `telemetry_surface.py` 上关闭 `coordinator.client` ghost seam
2. `25.2-02`：把现有 consumer tests 与 planning truth 同步到正式 `protocol` surface
3. `25.2-03`：回写 phase closeout truth、verification evidence 与 next-focus handoff

## Current Defect Chain

### 1. Telemetry bridge still reads legacy `client`

`custom_components/lipro/control/telemetry_surface.py` 仍通过 `coordinator.client` 构建 protocol telemetry source；而 concrete coordinator 已在正式主链上只承认 `self.protocol`。

### 2. Runtime type contract is no longer honest enough

`custom_components/lipro/runtime_types.py` 当前没有把 bridge 实际依赖的 `protocol` / `telemetry_service` 纳入 `LiproCoordinator` formal runtime public surface，导致 helper 只能走隐式属性探测。

### 3. Tests still legalize the ghost seam

`tests/integration/test_telemetry_exporter_integration.py` 与 `tests/core/test_system_health.py` 仍以 `client=` 构造 runtime coordinator 假对象。这等于在回归层把 legacy seam 继续合法化。

### 4. Planning truth still documents the seam as an active residual

`.planning/codebase/STRUCTURE.md`、`.planning/baseline/PUBLIC_SURFACES.md`、`.planning/baseline/VERIFICATION_MATRIX.md`、`.planning/reviews/RESIDUAL_LEDGER.md` 需要同步为“formal surface 已闭合、derived maps 只是协作图谱、residual 已关闭/收口”的单一叙事。

## Recommended Runtime Contract

### A. Formal telemetry source binding

- `LiproCoordinator` 必须显式暴露 bridge 所需的最小 `protocol` / `telemetry_service` surface。
- `telemetry_surface.py` 只从 `coordinator.protocol` 构造 protocol telemetry source。

### B. No legacy fallback

- `client` 只能作为历史术语存在于非本 phase touched seam 或 concrete implementation 细节中。
- control-plane telemetry bridge 不再接受 `client`-only coordinator。

### C. Governance honesty

- baseline / review / codebase maps 必须同步说明：formal source binding 已闭合；derived maps 不变更 authority 身份；本 phase 不重做 exporter schema。
