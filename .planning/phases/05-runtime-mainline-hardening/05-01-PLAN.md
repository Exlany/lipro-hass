# Plan 05-01: 收紧 runtime public surface 与 orchestration boundaries

## Preconditions

已满足：
- MQTT 单实例主链已落地；
- `Coordinator` 已具备 service-oriented runtime surface；
- `LiproProtocolFacade` 已成为统一协议根。

## Tasks

1. **正式化 runtime signal ports**
   - 将 connect-state / group-reconciliation 从 coordinator 私有回调提升为 formal signal ports；
   - 让 runtime wiring 依赖 service object，而不是 service object 依赖 coordinator 私有方法名。

2. **正式化 group reconcile owner**
   - 由 `CoordinatorDeviceRefreshService` 继续承接 group reconciliation 请求入口；
   - 通过 `Coordinator` 的 canonical refresh path 统一执行设备快照重建与订阅同步。

3. **收紧 wiring / boundaries**
   - 去除剩余的临时隐式 seam；
   - 保持 `Coordinator` 是唯一编排根，`MqttRuntime` 只发信号不夺权。

4. **同步 runtime telemetry surface**
   - 保持 connect-state / group-reconciliation 只经 `CoordinatorTelemetryService` 暴露；
   - 为后续 invariant tests 与 assurance guards 提供正式 snapshot。
