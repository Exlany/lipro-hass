## Verification Criteria

- baseline/docs/guards 对 host-neutral nucleus 讲的是同一条故事：formal helper/contract family，而非新 root；
- architecture policy 能同时阻断 `homeassistant` imports、`helpers.platform` 反向回流，以及字符串级 HA platform projection 在 nucleus homes 复活；
- `scripts/check_architecture_policy.py` 与 meta tests 已认识新增 rule ids / bans，不存在 policy 文档更新了但执行链不认的断层；
- residual/review docs 已记录任何 transitional seam 与 delete gate，不存在 silent carry-forward；
- focused meta + regression tests 能在 Phase 18 之后第一时间发现 locality/backflow 回归。

## Must-Haves

- 不得让 governance 文案暗示 `core` 已经变成独立跨平台 SDK
- 不得把 replay/evidence/exception hardening 冒充为本 plan 的主目标
- 不得只写文档不写 guard，也不得只写 guard 不同步 public-surface / review truth
- 所有规则都必须来自 north-star/baseline，同步到单一 enforcement chain，而不是新造第二套规则真相
