# Phase 7.5 Architecture

## Architectural Goal

把 `v1.1` 新增的 boundary / enforcement / telemetry / replay 资产收束进同一条 governance truth chain，使 closeout 由“分散文档 + 零散测试”升级为“可仲裁、可索引、可移交”的正式闭环。

## Single Truth Chain

```text
NORTH_STAR_TARGET_ARCHITECTURE.md
  -> baseline docs
     - AUTHORITY_MATRIX.md
     - PUBLIC_SURFACES.md
     - VERIFICATION_MATRIX.md
  -> review ledgers
     - FILE_MATRIX.md
     - RESIDUAL_LEDGER.md
     - KILL_LIST.md
     - V1_1_EVIDENCE_INDEX.md
  -> automated checks
     - check_architecture_policy.py
     - check_file_matrix.py
     - tests/meta/*
  -> closeout outputs
     - 07.5-SUMMARY.md
     - updated STATE / ROADMAP / REQUIREMENTS
```

裁决：
- baseline docs 继续定义 authority / surface / verification；
- review docs 继续承载 owner / residual / delete gate / evidence index；
- 自动化 checks 只执行真源，不自创第二套口径；
- closeout summary 只汇总已被治理真源与验证证据支持的结论。

## Formal Components

### 1. Governance matrices

正式职责：
- `AUTHORITY_MATRIX.md`：回答“谁拥有真源”；
- `PUBLIC_SURFACES.md`：回答“谁是正式 surface”；
- `VERIFICATION_MATRIX.md`：回答“哪条 requirement 由哪些证据证明”。

### 2. Review ledgers

正式职责：
- `FILE_MATRIX.md`：登记文件级 owner / phase / plane；
- `RESIDUAL_LEDGER.md`：显式列出未收口项与 delete gate；
- `KILL_LIST.md`：记录 residual 最终删除门；
- `V1_1_EVIDENCE_INDEX.md`：为工具/AI/维护者提供机器友好 closeout 指针。

### 3. Guard layer

正式职责：
- `check_architecture_policy.py` / `check_file_matrix.py`：本地与 CI 的快速裁决入口；
- `tests/meta/*`：对治理真源最易回归的部分做 focused regression proof。

## Cross-Phase Arbitration

### `07.3`

`07.5` 不重新定义 telemetry fields / redaction / cardinality；
只登记 exporter family 的 owner、证据与 residual。

### `07.4`

`07.5` 不改 replay corpus / deterministic driver 设计；
只登记 replay manifests / run summaries / meta guards 的 authority 与 evidence。

### `08`

`07.5` 不实现 evidence pack exporter；
只为 `08` 输出正式可 pull 的 `V1_1_EVIDENCE_INDEX.md` 与一致化治理指针。

## Closeout Deliverables

- v1.1 新资产的 owner / authority / verification / residual 全表对齐；
- v1.1 evidence index；
- phase summary 与 next-step recommendations；
- `STATE / ROADMAP / REQUIREMENTS` 的 planning-to-execution handoff。

## Non-Goals

- 不新增 production root；
- 不用文档“掩盖”尚未通过的验证；
- 不在 `07.5` 越权实现 `08` 的 evidence pack toolchain。
