# Plan 67-01: converge telemetry models, exporter views, and control bridges onto honest JSON-safe contracts

<objective>
把 telemetry failure-summary / sink / exporter / control bridge 一次性压回现有 formal contracts（`FailureSummary`、`TelemetryJsonValue`、exporter snapshot/views），并同步修复 telemetry / replay tests 对这些契约的消费方式；不允许通过 `dict[str, object]`、`Mapping[str, object]`、`Any` 或 `cast`-only 包装来消音。
</objective>
