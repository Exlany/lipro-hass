## 关键裁决

- **唯一根裁决**：`LiproProtocolFacade` 是 protocol plane 的唯一正式 public entry；`LiproRestFacade` / `LiproMqttFacade` 不得再被上层当作正式根。
- **normalization owner 裁决**：canonical normalized contracts 由 protocol root boundary 拥有，不允许继续散落在 REST / MQTT 各自侧边实现中。
- **目录裁决**：统一根的逻辑归属固定为 `custom_components/lipro/core/protocol/`；`core/api/` 与 `core/mqtt/` 只承载 child-façade 与 transport helpers。
- **残留裁决**：Phase 2.5 的清理目标不是“删 MQTT helpers”，而是删除 split-root public semantics 与 formal-root 误导命名。
