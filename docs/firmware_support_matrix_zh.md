## 验证流程

1. 在设备的 `update` 实体中查看是否有可用固件更新。
2. 通过 `update` 实体执行升级（未认证固件可能需要在确认窗口内再次点击安装以确认）。
3. 在诊断报告中记录升级前/后的状态。
4. 同步更新本页与 `custom_components/lipro/firmware_support_manifest.json`。
