## 性能对比分析

### 操作速度排名（从快到慢）

1. **设备键规范化**: 108.6 ns (基准 1.0x)
2. **运行时刷新检查**: 174.9 ns (1.61x)
3. **设备查找**: 238.5 ns (2.20x)
4. **设备注册**: 326.0 ns (3.00x)
5. **批量查找（单设备）**: 240.7 ns (2.22x)
6. **批量注册（单设备）**: 201.2 ns (1.85x)

### 关键发现

1. **DeviceIdentityIndex 高效**: 单设备操作在 200-350ns 范围内，适合高频访问
2. **批量操作优化良好**: 批量注册的单设备开销（201ns）低于单次注册（326ns）
3. **运行时决策轻量**: DeviceRuntime 的刷新检查仅需 175ns，不会影响主循环
4. **无明显性能瓶颈**: 所有测试操作均在微秒级以下完成
