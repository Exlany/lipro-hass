# Support

## English

### Before Asking for Help

Start with [README.md](README.md) for installation and setup, then check
[docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md) for common issues.

### Where to Go

| Need | Use |
| --- | --- |
| Usage question | GitHub Discussions if available, otherwise open an issue with context |
| Confirmed bug or regression | GitHub Issues |
| Security-sensitive report | [SECURITY.md](SECURITY.md) |
| Development or architecture question | [CONTRIBUTING.md](CONTRIBUTING.md) and [docs/README.md](docs/README.md) |

### What to Include

- Lipro integration version
- Home Assistant version
- Installation method
- Clear reproduction steps
- Expected and actual behavior
- Redacted logs or Home Assistant diagnostics

Never post passwords, tokens, private security findings, or unredacted personal
data in a public issue.

## 中文

### 提问前

安装与配置请先看 [README_zh.md](README_zh.md)，常见问题请先看
[docs/TROUBLESHOOTING.md](docs/TROUBLESHOOTING.md)。

### 去哪里反馈

| 需求 | 入口 |
| --- | --- |
| 使用问题 | 如仓库已开启 Discussions，优先使用 Discussions；否则请带上下文创建 Issue |
| 已确认的缺陷或回归 | GitHub Issues |
| 安全敏感问题 | [SECURITY.md](SECURITY.md) |
| 开发或架构问题 | [CONTRIBUTING.md](CONTRIBUTING.md) 和 [docs/README.md](docs/README.md) |

### 建议提供的信息

- Lipro 集成版本
- Home Assistant 版本
- 安装方式
- 可复现步骤
- 预期行为与实际行为
- 已脱敏的日志或 Home Assistant 诊断信息

请勿在公开 Issue 中粘贴密码、Token、安全敏感细节或未脱敏的个人数据。
